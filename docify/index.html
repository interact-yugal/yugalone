Docify

Docify bridges the gap between Google Sheets and Google Docs to produce consistent documents. We help teams move beyond manual copy-pasting into a world of structured, repeatable document workflows.

Connect your data source, selection logic, and a Google Docs template into a repeatable workflow to generate high-fidelity documents in bulk. Save workflows, share with your team, create documents-all within Google Workspace.

Core Capabilities

1. Advanced Workflow Management

  • Save & Load States: Persist complex configurations (sheet, template, settings) as named workflows for instant reuse.
  • Collaborative Sharing: Share workflows with specific users via email; includes role-based access (Owners can modify/delete; Shared users can only load).
  • Version Control: Automatic versioning of workflows with real-time update banners notifying users if the owner has updated the currently loaded workflow.
  • Multi-Tenancy Storage: Secure, user-specific storage of workflow indexes and data.

2. Data Intelligence & Control

  • Seamless Integration: Native Google Picker integration for secure selection of Google Sheets and Drive Folders.
  • Data Hygiene Tools: Built-in utilities to clean data sources by removing duplicate rows, blank rows, and blank columns before processing.
  • Granular Record Selection: Search, filter, and select specific records in real-time, or use the detailed "Card View" to inspect individual record data.

3. Smart Template Engine

  • Template Analysis: Automatically scans templates to calculate complexity, character count, and estimated pages.
  • Dynamic Limits: Recommends "Safe Record Limits" (e.g., max records per batch) based on template complexity to prevent timeouts.
  • Auto-Generator: One-click creation of a new Google Docs template populated with placeholders derived directly from the selected Sheet headers.
  • Flexible Syntax: Supports placeholder formats like <<[Field Name]>> or <<FieldName>>.

4. Flexible Document Generation

  • Dual Structures: Choose between "Individual Files" for separate documents per record or "Single File" to merge all records into one continuous document.
  • Multi-Format Output: Generate as Google Docs, PDF, or Docs + PDF.
  • PDF Engine: Uses Advanced Drive Service (v3) for high-fidelity PDF conversion with a "Trash Original" option to keep folders clean.
  • Dynamic Naming: Custom file naming patterns using placeholders (e.g., Invoice - <<[ClientName]>>).

5. Performance & Reliability

  • Quota Management: Automatically detects account type (Gmail vs. Google Workspace) and applies appropriate daily processing limits.
  • Rate Limiting: Intelligent batch processing with delays to prevent Google API "Too Many Requests" errors.
  • Execution Safety: Monitors execution time and safely stops processing before script timeouts occur.
  • Robust Error Handling: Detailed error logging and user-friendly notifications for permissions or access issues.

6. Modern User Experience

  • Smooth Interface: Implements Lenis for smooth scrolling and a polished, tabbed UI layout.
  • Keyboard Accessibility: Full keyboard support for navigation, including shortcuts for closing modals (Esc), confirming actions (Enter), and deleting workflows (Delete).
  • Interactive Feedback: Loading overlays, success/error toast notifications, and complexity rating indicators for templates.

Designed for teams that need reliable, repeatable automation workflows. Deploy in minutes and scale your document operations without the overhead.
